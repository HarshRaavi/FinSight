# FinSight – Intelligent Profile Evaluation System

FinSight is a full-stack application that evaluates profiles using structured and unstructured data and produces ranked results.

## Tech Stack

Frontend
- React
- Axios

Backend
- Spring Boot
- REST APIs
- JPA

## Features

- Profile submission
- Resume analysis
- Automated scoring
- Ranked candidate evaluation

## Architecture

React UI → Spring Boot API → Evaluation Engine → Database
